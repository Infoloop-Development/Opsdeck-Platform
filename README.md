# OpsDeck | A Complete Project Management Solution

OpsDeck is a robust and user-friendly project management application built with [Next.js](https://nextjs.org/) and [Material-UI (MUI)](https://mui.com/). Designed to help teams streamline workflows, manage tasks effectively, and boost productivity.

---

## Features

### Core Features

- **Dashboard**: Intuitive dashboards with real-time metrics and KPIs.
- **Task Management**: Create, assign, and monitor tasks with deadlines and progress tracking.
- **Reports Module**: Generate detailed insights for data-driven decisions.
- **Notifications**: Real-time updates to stay informed.
- **Contracts**: Manage and track project-related contracts.
- **Feedback**: Collect and analyze feedback for improvements.
- **Profile Settings**: Advanced customization options for user profiles.
- **Dark Mode & Theme Switching**: Seamlessly toggle between light, dark, and custom themes.
- **Charts & Visualizations**: Leverage ApexCharts and Gantt charts for detailed progress views.

### Built-in Pages

- **Home**: A welcoming landing page.
- **About**: Insight into the app and its purpose.
- **Privacy Policy**: Transparent user data policy.
- **Careers**: Explore career opportunities with the team.

---

## Tech Stack

- **Frontend**: [Next.js](https://nextjs.org/) with App Router for modern, efficient routing.
- **UI Framework**: [Material-UI](https://mui.com/) for responsive, customizable components.
- **State Management**: Redux for efficient state handling.
- **Charts & Visualizations**: [ApexCharts](https://apexcharts.com/) and [Frappe Gantt](https://frappe.io/gantt).
- **TypeScript**: Strongly-typed code for reliability and clarity.

## Installation

### Prerequisites

Ensure you have the following installed:

- Node.js (v18 or later)
- npm or yarn

### Steps to Run the Application

1. Clone the repository:

   ```bash
   git clone your-repo-link.git
   cd project-management-app
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server:

   ```bash
   npm run dev
   # or
   yarn dev
   ```

   Access the app at `http://localhost:3000`.

4. Build for production:

   ```bash
   npm run build
   npm run start
   ```
